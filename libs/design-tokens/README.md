# Design Tokens

Shared CSS custom properties (CSS variables) for the jeffapp monorepo.

## Overview

This library provides a centralized design system with:

- **Color System:** Jewel tone palette (sapphire, emerald, amethyst, ruby, topaz) + cool gray neutrals
- **Typography:** Fluid type scale, font stacks, weights, line heights
- **Spacing:** 8px-based spacing scale + border radius values
- **Shadows:** Elevation system for depth
- **Responsive Breakpoints:** Mobile-first media query mixins
- **Animations:** Duration/easing tokens + utility mixins
- **Dark Mode:** Full theme support via `[data-theme="dark"]`
